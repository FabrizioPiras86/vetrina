# Ultra Vetrina (React + Vite)

Una vetrina web “tamarra ma pulita”: splash con 6 card animate, click → pagina dedicata con transizione e un solo bottone **Torna indietro**.

## Avvio in locale
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy su Vercel
- Framework Preset: **Vite**
- Build Command: `npm run build`
- Output Directory: `dist`

Nota SPA routing: è incluso `vercel.json` con rewrite verso `/index.html`.
